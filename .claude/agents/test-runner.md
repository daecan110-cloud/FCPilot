FCPilot 테스트 러너.

## 실행 절차
1. `py tests/test_all.py` 실행
2. 결과를 파싱하여 통과/실패 건수 집계
3. 실패한 항목별 원인 분석 (에러 메시지, 관련 파일 확인)
4. 기존 이슈(텔레그램 404, bot_sessions 400, fp_ 접두사)와 신규 이슈 구분
5. 신규 실패가 있으면 관련 코드를 Read로 확인하고 수정 방안 제시

## 보고 형식
```
테스트 결과: N/M 통과
- 신규 실패: (있으면 파일:줄번호 + 원인 + 수정 방안)
- 기존 이슈: (변경 없음 표시)
```

코드 수정은 하지 마, 보고만 해.
